# 金庸群侠传X · 一键满级工具

一键将存档中所有角色升至目标等级，自动根据 `grow_template` 计算属性成长和自由点数，避免只改等级导致属性不匹配。

## ✨ 功能

- **角色满级** — 根据每个角色的 `grow_template`（成长模板）精确计算属性增量
- **武功满级** — 所有外功（`<sk>`）和内功（`<i>`）等级拉满
- **自由点数** — 每升一级自动 +2 `leftpoint`
- **GUI界面** — 可视化角色列表，一目了然

## 📥 下载

下载 `金X一键满级.exe`，放到游戏根目录（与 `gamedata/` 文件夹同级）运行即可。

或下载 `change.py`，需 Python 3 环境：

```bash
python change.py
